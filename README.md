# PENGEMBANGAN-SISTEM-EVALUASI-CERDAS-USULAN-JUDUL-TESIS
Peningkatan jumlah mahasiswa Magister Teknik Informatika berdampak pada  lonjakan pengajuan usulan judul tesis. Evaluasi manual yang dilakukan oleh  program studi seringkali memakan waktu dan rentan terhadap subjektivitas,  sehingga memunculkan risiko disetujuinya topik penelitian yang usang atau kurang  memiliki kebaruan (novelty).
